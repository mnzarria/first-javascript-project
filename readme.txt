Edit 4/11/2022

La página 'Elegí tus plantas' va a tener un formulario que permita detectar las plantas adecuadas para la casa del usuario. El usuario podrá buscar plantas según la cantidad de luz que tenga en su casa, la frecuencia de riego que esté dispuesto a realizar, los cuidados que requiera la planta (dificultad), el precio que pueda gastar, o directamente por nombre. Al completar una búsqueda con resultados, se mostrará en pantalla la imagen y todos los datos de cada planta que cumpla con los requisitos.

El botón Crear planta va a llamar a la función del mismo nombre, y habilitará un formulario de carga de datos, que permitirá Cargar plantas nuevas a la base de datos (array).

Actualmente es posible primero cargar una planta y luego con la busqueda comprobar que la planta se ingresó a la base de datos. 

Posibles features:
-Incluir recomendaciones para cada planta (donde ubicarla, si es para colgar o no, si requiere un tutor, que sustrato lleva, etc)
-Recomendar planta al azar, utilizando math.random con el id de cada planta.
-Vincular las plantas con los modelos de maceta
